Proyecto - API hecho en Nestjs y Postgres, para la administración de pedidos, 

Los clientes son, una web hecho en Nextjs, y una app hecho en Flutter.

Incluye:

- Control de Stock
- Compras
- Ventas
- Gestion de Pedidos
- Facturacion
- Móviles
- Reparto
- Informes

Tecnologias:

- JWT
- TypeScript
- TypeORM
- Postgres
- Docker
- Express
